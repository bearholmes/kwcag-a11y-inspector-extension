소개

  • KWCAG 2.1의 2.1.3 조작 가능 검사항목을 측정하기 위한 기능 제공
  • 1.3.3 텍스트 콘텐츠의 명도 대비 검사항목을 측정시 도움될 수 있는 보조 기능 제공
  • 기준 : box + padding + border
tip
  • * 주의사항 : 아쉽게도 크롬의 특성상 iframe, frame 내부 문서는 확인할 수 없습니다.
  • * 확장프로그램 아이콘을 우클릭하면 수동계산을 하실 수 있습니다.

단축키 안내

  • 단축키를 이용하면 좀 더 편리하게 이용할 수 있습니다.
    • Alt + Q : 실행 / 정지 (기본값)
    • Esc : 일시정지 / 재개 (활성화된 상태에서 가능)
  • 실행 / 정지 단축키는 다음의 방법으로 변경할 수 있습니다.
    • 1. chrome://extensions/로 이동
    • 2. 확장프로그램 목록 최하단에 "단축키" 링크 선택
    • 3. "dk a11y inspector - 확장 프로그램 활성화" 항목에서 단축키 설정
